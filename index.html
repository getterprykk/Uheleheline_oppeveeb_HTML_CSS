Mis on HTML?

HTML ehk Hypertext Markup Language on standardne märgistuskeel veebilehtede ja veebirakenduste loomiseks. HTML-i kasutatakse veebilehel kuvatava sisu struktureerimiseks ja vormindamiseks ning seda saab kasutada piltide, videote ja muu multimeediumi lisamiseks

Teooria

HTML tähendus.

HTML (ingliskeelne lühend sõnadest HyperText Markup Language 'hüperteksti märgistuskeel') on keel, milles märgendatakse veebilehti. Versioonid 2.0 kuni 4.01 HTML olid SGML-i rakendused. HTML-dokumentide (failide) veebist allalaadimiseks ja vaatamiseks kasutatakse enamasti veebibrauserit.

Olulised koodid

Terminalis projekti muudatuste üleslaadimine github'i

git add .

git commit -m " mida muutsid"

git push

Muudatuste alla tõmbamine github'ist

git pull

Oluline

header – lehe või sektsiooni päis. Tavaliselt sisaldab logo, pealkirja või navigeerimist.

nav – spetsiaalne ala, kus paiknevad navigeerimislingid. Ekraanilugejad tunnevad selle ära kui navigeerimise bloki.

section– temaatiline sisublokk, millel on oma pealkiri (h2 jne). Näiteks „Meist“, „Teenused“.

article – iseseisev sisutükk, mis võiks ka üksinda avaldatuna tähendust omada (uudis, blogipostitus).

aside – lisainfo, nt küljeriba või viited seotud sisule.

footer – lehe või sektsiooni jalus (autoriõigused, kontaktid).

Lehe paigutus

Flex box

Mõeldud ühe mõõtme jaoks (rea või veeru paigutus).

Sobib horisontaalse või vertikaalse elemendirea joondamiseks.

Oluline omadus: display: flex;.

Põhilised sätted: justify-content (joondus reas), align-items (joondus veerus).

Grid

Mõeldud kahe mõõtme jaoks (ridade ja veergude paigutus).

Sobib keerukamate lehtede ja kaartide paigutamiseks.

Oluline omadus: display: grid;.

Põhilised sätted: grid-template-columns, grid-template-rows, gap.
